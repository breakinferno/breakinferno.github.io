2. markdown 介绍

Markdown是一种轻量级的「标记语言」,赋予了普通文本文件格式化的功能，同时又保持了文本文件的简洁。

优点

  • Markdown简单易用、可读、直观、学习成本低。
  • 纯文本，兼容性极强，可以用所有编辑器打开
  • 让你专注文字而不是排版
  • 格式转变方便，可以转为pdf,html,电子书等
  • 标记语法具有很好的可读性

3. 工具选取

markdown有很多编辑器或者在线编辑器。这里推荐几个。

在线编辑网站

  1. Cmd Markdown编辑器:作业部落出品，也是一款不错的工具和博客平台兼顾的产品。全平台且提供web版
  2. MaHua 在线Markdown编辑器:小众软件推荐，界面有些简陋
  3. dillinger Markdown编辑器:漂亮强大，支持md, html, pdf 文件导出。支持dropbox, onedrive，google drive, github. 来自国外，可能不够稳定.
  4. 简书:一个很不错的博客平台，每几秒钟便会自动存入一个备份。可以直接从本地拖入照片生成链接，一直在不断优化。作为一个博客平台，需要注册账号后方能进行写作
  5. 马克飞象:跟印象笔记有关，没用过。
  6. 小书匠编辑器:全平台覆盖并且有web版

本地编辑器软件

  1. sublime text:轻量强大的编辑器，下载插件即可使用(ps:ctrl+shift+p install package;下载markdown Editing 和 markdownlivepreview插件即可)

这里是吕飞的博客



4.3.2 参考式


参考式主要用于学术论文上面，或者某个链接在某篇文章中多次引用，这种情况使用参考式就十分舒服，可以让你对链接进行统一管理。

语法说明

  参考式分为两个部分，文中链接部分使用[链接文字][链接标记]表示，在其他任意位置使用[链接标记]:链接地址 "链接标题"表示，注意在链接标题前有一个空格。

  还有另外一种情况,如果链接文字和链接标记相同，则可以在链接部分简化链接标记为[]。话不多说，来看实例吧。

代码

code

1.这里有几个网站：[百度][1]、[哔哩哔哩][2]、[吕飞的博纳阁][3]
2.这个是第二种情况：[Net][]
3.
4.[1]:http://baidu.com
5.[2]:http://bilibili.com
6.[3]:http://breakinferno.info
7.[Net]:http://example.com



效果

result

这里有几个网站：百度、哔哩哔哩、吕飞的博纳阁
这个是第二种情况：Net
